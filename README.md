ШПАРГАЛКА ПО МАТЕМАТИКЕ (4 семестр, ИТКН)

========================================
1. ФОРМУЛА ПОЛНОЙ ВЕРОЯТНОСТИ
========================================
P(A) = Σ P(H_i) * P(A|H_i)
Гипотезы H_i — несовместны и дают все варианты.

Пример: из 1-й урны переложили шары во 2-ю, затем вынули шар.
→ найти P(H_i) через комбинаторику
→ найти P(A|H_i) по новому составу
→ сложить

========================================
2. СХЕМА БЕРНУЛЛИ
========================================
P_n(k) = C_n^k * p^k * q^(n-k),  q = 1-p

C_n^k = n! / (k! * (n-k)!)

Памятка: 0! = 1

========================================
3. ГЕОМЕТРИЧЕСКАЯ ВЕРОЯТНОСТЬ
========================================
P = площадь области / площадь всего пространства

Два числа из [0;1], условие x+y≤1, xy≥a:
Площадь = ∫(1-x - a/x)dx от x1 до x2,
где x1, x2 — корни x(1-x)=a

========================================
4. НЕПРЕРЫВНАЯ СВ (ПЛОТНОСТЬ f(x))
========================================
F(x) = ∫_{-∞}^{x} f(t) dt  (интегральная функция)

M(X) = ∫ x f(x) dx

D(X) = ∫ x² f(x) dx - [M(X)]²

Проверка: ∫ f(x) dx = 1 (по всей прямой)

========================================
5. ДВУМЕРНАЯ ДИСКРЕТНАЯ СВ
========================================
M(X) = Σ x_i * P(X=x_i)
M(Y) = Σ y_j * P(Y=y_j)
M(XY) = ΣΣ x_i y_j * p_{ij}

Ковариация: cov = M(XY) - M(X)M(Y)

Дисперсия: D(X) = M(X²) - [M(X)]²
          M(X²) = Σ x_i² * P(X=x_i)

Коэффициент корреляции: ρ = cov / √(D(X)D(Y))
|ρ| ≤ 1

========================================
6. НОРМАЛЬНОЕ РАСПРЕДЕЛЕНИЕ N(μ, σ)
========================================
Плотность: f(x) = 1/(σ√(2π)) * e^{-(x-μ)²/(2σ²)}

Стандартизация: Z = (X-μ)/σ ~ N(0,1)

P(a<X<b) = Φ((b-μ)/σ) - Φ((a-μ)/σ)
Φ(x) — функция Лапласа (таблица)

Правило трёх сигм: P(|X-μ|<3σ) ≈ 0,997

========================================
7. МАТЕМАТИЧЕСКАЯ СТАТИСТИКА
========================================
Выборка x₁...xₙ:

Оценка M:  x̄ = (1/n) Σ x_i

Оценка дисперсии (несмещённая):
s² = 1/(n-1) * Σ (x_i - x̄)²

Оценка СКО: s = √(s²)

========================================
8. ДОВЕРИТЕЛЬНЫЕ ИНТЕРВАЛЫ (γ=0,95)
========================================

ДЛЯ M (малая выборка, дисперсия неизв.) — Стьюдент:
x̄ ± t_{0,025; n-1} * s/√n

t_{0,025;5}=2,571   t_{0,025;9}=2,262

ДЛЯ σ — хи-квадрат:
σ ∈ ( s√((n-1)/χ²_{0,025}),  s√((n-1)/χ²_{0,975}) )

n-1=5:  χ²_{0,025}=12,832   χ²_{0,975}=0,831
n-1=9:  χ²_{0,025}=19,023   χ²_{0,975}=2,700

========================================
10. ЧЕК-ЛИСТ ПО ЗАДАЧАМ
========================================

Задача 1 (полная вероятность):
→ гипотезы → P(H_i) → P(A|H_i) → сумма

Задача 2 (Бернулли):
→ p для одного испытания → P(все) = p³ → P(ровно 1) = 3pq² → отношение

Задача 3 (непрерывная СВ):
→ найти c из ∫f=1 → F(x) → M(X) → D(X)

Задача 4 (двумерная):
→ дополнить таблицу (сумма=1) → маргинальные вероятности
→ M(X), M(Y), M(XY) → cov → D(X), D(Y) → ρ

Задача 5 (нормальное + Бернулли):
→ p = P(попасть в интервал) через Φ
→ P₃(2) = C₃² * p² * (1-p)

Задача 6 (статистика):
→ x̄ → s², s → ДИ для M (Стьюдент) → ДИ для σ (хи-квадрат)

========================================
11. ТЕОРЕТИЧЕСКИЕ ВОПРОСЫ (УСТНО)
========================================

Формула полной вероятности:
P(A)=ΣP(H_i)P(A|H_i), где H_i — полная группа событий.

Интегральная функция распределения:
F(x)=P(X≤x). Свойства: неубывает, от 0 до 1, F'(x)=f(x) для непрер. СВ.

Нормальное распределение:
N(μ,σ²). Плотность, стандартизация Z=(X-μ)/σ, правило 3σ.
